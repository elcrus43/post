# Room: Frontend

## Tech Stack
- **Framework**: React 18+
- **Bundler**: Vite
- **Styling**: Vanilla CSS (Tailwind excluded unless requested)

## Design Guidelines
- Modern, premium aesthetics.
- High interactiveness (hover effects, transitions).
- Dynamic design that feels "alive".

## Planned Features
- Analytics tracking dashboard.
- Real-time post previews.
- Mobile-responsive layout refinement.
- Mobile App (Android/iOS) version.
